# Subhramoy Portfolio

A Vite + React portfolio website for Subhramoy Bhowmik. It uses reusable React components, data-driven project and tech stack sections, GSAP animations, Lenis smooth scrolling, Bootstrap utilities, Font Awesome icons, and custom CSS.

## Scripts

- `npm run dev` starts the local Vite development server.
- `npm run build` creates a production build.
- `npm run preview` serves the production build locally.
